# IPC1_EJERCICIOS_202501915
## ¿Qué aprendí con esta tarea?

Con esta tarea aprendí a crear y configurar un repositorio público en Github comprendiendo la importancia del control de versiones en el desarrollo de software y organizar adecuadamente un proyecto. 

Ademas, entendi la importancia de documentar correctamente un proyecto mediante un archivo README.md, lo cual facilita que otras personas comprendan el funcionamiento del repositorio.

## ¿Cómo crees que aplicarás este conocimiento en otro curso o en la vida profesional?

Este conocimiento será fundamental en otros cursos relacionados con programación y desarrollo de software, ya que GitHub permite trabajar de manera organizada  en proyectos. 

En lo profesional, el uso de repositorios públicos y privados será esencial para trabajar en equipo y gestionar versiones de proyectos, GitHub es una herramienta clave en lo laboral, en áreas de desarrollo de software y proyectos
